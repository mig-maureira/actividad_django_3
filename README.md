# actividad_django_3

Instrucciones
A partir del trabajo visto en las presentaciones del módulo I, y continuando la línea de los visto en
el presente módulo, integrar en el proyecto hecho en Django.
Consideraciones:
- Aplicar principio DRY.
- Utilizar entorno virtual.
- Utilizar herencia de plantillas (Sugerencia, denominar a la primera como base.html, y que
contenga header, navbar y footer).
- Respetar el patrón de diseño MTV.
